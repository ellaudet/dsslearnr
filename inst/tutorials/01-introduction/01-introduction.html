Welcome and Instructions

This is a tutorial for readers of DSS, which stands for Elena Llaudet and Kosuke Imai’s Data Analysis for Social Science: A Friendly and Practical Introduction (Princeton University Press, 2022).

DSS teaches from scratch R, statistics, and the fundamentals of quantitative social science (that is, the fundamentals of causal inference, survey research, and predictive models). The practice exercises you will find here cover all of these and are organized by chapter of the book.

Each chapter is divided into two to five topics, and each topic corresponds to one to five sections of the textbook. In other words, each topic reviews only a handful of pages of the textbook, which often can be read in a single seating. Here is the recommended sequence:

(B) Practicing Typing and Running Code

Instructions

When running code on our own computers, we write the code on the R script (the upper left window of RStudio), then we highlight the code we want to run and either (a) manually click on the Run icon of RStudio or (b) use the shortcut command+enter in Mac or ctrl+enter in Windows.

When running code on the tutorials, however, we have to do things a little differently. Every time an exercise asks you to type and run some code, you will see an R code window. This is where you will type the code.

Once you are done typing the code, you can run it by clicking on the Run Code button in green. The output produced by the executed code will appear underneath the R code window.

Once you are satisfied with your answer, you can submit it by clicking on the Submit Answer button in blue.

To help you figure out the right answer, some questions contain hints. You can access them by clicking on the Hint button in white.


Q1: Ask R to compute 30+6

Hint: You may want to use the + operator.


Q2: Ask R to store 30+6 into an object called thirtysix

Hint: You may want to use the assignment operator <-.

To the left of the assignment operator, we specify the name of the object (without quotes).

To the right of the assignment operator, we specify the contents of the object.

Recall: The name of objects cannot contain spaces.


Q3: Ask R to show the contents of the object thirtysix

Hint: To ask R to show us the contents of an object, we run the name of the object.


Q4: Use the sqrt() function to ask R to compute the square root of 36

Hint: The function sqrt() calculates the square root of the argument specified inside the parentheses.

Suppose we have already set the working directory to the DSS folder on our Desktop and we run the code below to read and store the STAR dataset in an object called star.

star <- read.csv("STAR.csv") # reads and stores data


Q1: Use the head() function to ask R to show the first six observations of the object star

Hint: The head() function shows the first six observations in a dataframe and the only required argument is the name of object where the dataframe is stored.


Q2: Now, use the head() function again to ask R to show the first eight observations of the object star

Hint: To change the number of observations displayed by the head() function, we can use the optional argument named n.

Recall: We do not use quotes around the names of arguments, and when we specify more than one argument, we need to separate them with a comma.


Q3: Now, use the dim() function to ask R what the dimensions of the object star are

Hint: The dim() function provides the dimensions of a dataframe. The only required argument is the name of the object where the dataframe is stored (without quotes). The output is two values: the first indicates the number of observations in the dataframe; the second indicates the number of variables.

head(star) # shows the first six observations


Q1: Use the mean() function to compute the mean of the variable reading, which is stored inside of the object star. (Hint: The $ character might be helpful here)

Hint: The mean() function calculates the mean of a variable. The only required argument is the code identifying the variable. To identify the variable, we can use the $ character.

We use the $ character to access a variable inside a dataframe. To its left, we specify the name of the object where the dataframe is stored (without quotes). To its right, we specify the name of the variable (without quotes). Example: data$variable.


Q2: Now, use the mean() function again, but this time to compute the mean of the variable graduated, which is also stored inside of the object star. (Hint: The $ character might be helpful here)

Hint: The mean() function calculates the mean of a variable. The only required argument is the code identifying the variable. To identify the variable, we can use the $ character.

We use the $ character to access a variable inside a dataframe. To its left, we specify the name of the object where the dataframe is stored (without quotes). To its right, we specify the name of the variable (without quotes). Example: data$variable.
